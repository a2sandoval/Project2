Portfolio Projects

I've been doing web-coding since early 2018, and have accumulated a lot of little projects over time. To help show my progression and to separate the simpler projects from the more complicated ones, I've divided the Portfolio into a few different sections. Look for the Github icon () for a link to the Github repo and look for the play icon () for a link to a deployed version, if one exists. Some of these are command line applications, so not every project will have a deployed version.

  • I. Early Projects
  • II. Adding in Back-End
  • III. Let's Combine Them

Early Projects

  • The Psychic Game. Hangman, made even simpler. Guess the letter the computer is thinking of.
  • Word Guess Game. Classic Hangman, themed arouned The Beatles' song titles.
  • Crystal Collector. Click on the crystals to get match a number. The twist - every round each crystal will have a new random value!
  • Star Wars: Ultimate Fighter. Choose your fighter and battle your way through the rest of the galaxy! As you fight more opponents you get stronger, so choose your opponents wisely. (See Above)
  • Trivia Game. How well do you know book and movie Lord of the Rings trivia? Test your knowledge with this quick game.
  • GifTastic. Search for various Gifs!
  • Mastermind. The classic guessing game of determing a set of colors based on how correct your guesses are. There are three levels and various themes to choose from.

Adding in Back-End

  • Liri Node JS bot. A first foray into the world of node.js. Uses a few different node packages to respond to user commands. This is run on the command line.
  • Command Line Word Guess Game. Hangman, but on the command line.
  • Bamazon. A starting effort into creating a program that uses MySQL. "Customers", "Managers", and "Supervisors" can interact with this online "store". This is a command line program.
  • Friend Finder. Uses a simple algorithm to pair a user to someone in the MySQL database based on how they answer a few questions. (See Above)
  • Burger Burger. Users can "add" and "eat" burgers listed on website. Implemented to practice using Handlebars.

Let's Combine Them

  • The What? An online character sheet for the Doctor Who Roleplaying Game. Users can view characters others have created, as well as viewing their own character. (See Above)
